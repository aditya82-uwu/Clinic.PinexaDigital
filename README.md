# Harborview Family Clinic — Demo Site (clinic)

A standalone Next.js demo site for the medical/clinic industry. Cloned from
the gym demo as a working example of the "use this template" workflow.
Deploy as its own Vercel project, then attach `clinic.pinexadigital.com`
under that project's Settings → Domains.

## Running locally
```bash
npm install
npm run dev
```

## Editing content
Everything editable lives in `src/lib/site-content.ts`.

- Same `noindex` + contact form notes as the gym demo's README apply here.
